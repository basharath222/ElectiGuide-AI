# ElectiGuide AI 🗳️

An autonomous agent that doesn't just "talk" about elections but acts for the user. It creates personalized election roadmaps, generates Calendar reminders, and finds polling stations using a blazing-fast, 100% Zero-Backend static architecture.

## Chosen Vertical
**Election Process Education**
ElectiGuide AI is designed to help users navigate the complex landscape of voting laws, registration deadlines, and polling locations. It aims to reduce friction in the civic participation process by offering localized, actionable intelligence and seamlessly integrating with the user's digital life.

## Zero-Backend Architecture
This project is built purely with HTML, CSS, and Vanilla JS to guarantee a footprint under 1MB. There is no Node.js backend, no build steps (webpack/vite), and no NPM dependencies.

1. **Gemini 1.5 Flash (Direct REST API)**: 
   - *Role*: Powers the natural language conversational agent and extracts calendar dates. Primed to act as a Senior Civic Tech Expert specializing in localized election law based on the user's zip code.
   - *Implementation*: Direct `fetch()` call to the `v1beta/models/gemini-1.5-flash-latest:generateContent` REST endpoint.
2. **Google Maps Integration**:
   - *Role*: Locates polling stations and voter registration offices visually.
   - *Implementation*: Uses the Google Maps `<iframe>` Embed API based on zip code, completely avoiding complex Places API server routing.
3. **Google Calendar Smart Links**:
   - *Role*: 'Sync My Voter Journey' feature.
   - *Implementation*: Instead of a heavy OAuth flow, the app dynamically constructs a Google Calendar `TEMPLATE` URL with dates generated by Gemini, allowing users to add events to their calendar in a single click without exposing server-side permissions.

## Logic Flow (Contextual UI Wizard)
The interface uses a fast, modular State object in `app.js`:
1. **Location**: Ask for ZIP Code to anchor the search.
2. **Eligibility**: Capture Age and Registration Status.
3. **Roadmap**: Generate the Map `<iframe>` and Calendar links dynamically.
4. **Contextual Chat**: The user can open a chat session with Gemini 1.5 Flash, system-primed with their location to provide highly accurate legal answers.

## Running Locally

Because this is a static site, you don't even need Node.js!

1. Open `config.js` and add your API Keys:
   ```javascript
   const CONFIG = {
       GEMINI_API_KEY: 'your_key_here',
       MAPS_API_KEY: 'your_key_here'
   };
   ```

2. Simply open `index.html` in your browser!
   *(Or serve it with any lightweight static server like Python's `http.server` or `npx serve`)*
